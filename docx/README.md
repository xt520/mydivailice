# Alice 使用文档（docx）

这是一套面向“第一次接触 Alice v0”的使用文档，目标是让你按顺序阅读即可跑通示例并理解语言的主要能力。

6. [05-LSP（编辑器智能提示）](./05-LSP.md)
7. [06-打包与发布](./06-打包与发布.md)

## 约定

- 所有网络示例只使用 `127.0.0.1`，并尽量使用 `port=0` 自动分配端口。
- 示例均可在 `examples/` 与 `test/` 中找到；推荐优先运行 `./alice selftest` 做回归验证。
