# 00-快速上手

目标：用最少命令把 Alice 程序跑起来。

## 1. 构建

在仓库根目录：

```bash
dotnet --version
dotnet build "Alice.sln"
```

## 2. 运行（推荐使用启动器）

仓库根目录提供了薄启动器脚本，避免每次输入 `dotnet run --project src/alicec -- ...`。

- Linux/macOS：`./alice`（或旧脚本 `./alicec.sh`）
- Windows：`alice.cmd` 或 `./alice.ps1`

示例：

```bash
# 直接运行（等价于 run）
./alice examples/bubble_sort.alice

# 显式 run
./alice run examples/bubble_sort.alice

# 自测
./alice selftest
```

## 3. 运行带 unsafe 的代码

如果代码中使用了 `@unsafe`（指针/NativeBuffer 相关），必须显式开启 `--allow-unsafe`：

```bash
./alice run examples/unsafe/PtrU8Write.alice --allow-unsafe
```

## 4. 常用命令速查

```bash
./alice check <file.alice>
./alice run <file.alice>
./alice build <file.alice> -o <out.dll>
./alice bind --ref <path.dll> -o <outDir>
./alice selftest

# 打包
./alice package <file.alice> --mode both

# 启动 LSP（Language Server）
./alice lsp
```
