# 04-网络与 IO

## 1. 文件 IO

示例：`examples/io/FileRoundtrip.alice`、`examples/io/DeferClose.alice`

## 2. TCP/UDP

示例：`examples/net/TcpEcho.alice`、`examples/net/UdpEcho.alice`

## 3. HTTP client/server

示例：`examples/http/HttpServerClient.alice`

提示：`srv.serveOnce(host, port, handler)` 返回实际监听端口；推荐 `port=0` 自动分配。

## 4. TLS

当前 `net.tls` 提供 `wrap/dial/Smoke`（最小封装/占位 smoke）。
如果你只需要验证模块可用，运行 `examples/tls/Smoke.alice`。
