# 02-并发与异步

## 1. thread 与 channel（sync）

- 线程：`thread.spawn` 返回 `JoinHandle`，用 `Join()` 等待。
- channel：`sync.makeChan<T>(cap)`，`cap=0` 表示无缓冲。

示例：`examples/sync/ChanPingPong.alice`

## 2. async/await/go

- `async fun` 会发射到 C# `Task/Task<T>`。
- `await expr`：等待 Task。
- `go callExpr`：用 `Task.Run` 在后台执行调用。

示例：

- `examples/async/AsyncFun.alice`
- `examples/async/GoThen.alice`

## 3. Then/Catch/Finally 与 withTimeout

标准库 `async` 提供 `Then/Catch/Finally/withTimeout`。

示例：`examples/async/Timeout.alice`

注意：如果函数标了 `@unsafe`，则生成的 C# 会处于 `unsafe` 上下文；C# 不允许在 `unsafe` 上下文里 `await`。
