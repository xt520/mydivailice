# 01-语言速览

本篇只覆盖“能写程序”的最小语法与最常用能力。更完整能力请继续读后续章节。

## 1. 模块：namespace / import

```alice
namespace demo

import io as io
```

## 2. 变量声明

- 局部推导：`x := expr`
- 局部显式类型：`x: int = expr`

注意：顶层禁止 `:=` 推导，必须使用显式类型（或 `const`）。

## 3. 函数

## 7. 显式类型转换（as）

Alice 支持显式强制类型转换：

```alice
x: int = 65
c := (x as char)
print(c)

y := (x as u8)
print(y)
```

当前实现直接发射为 C# 的显式 cast（`(T)(expr)`）。
示例见：`examples/cast/Basic.alice`。
